### Prerequisites

You only need a modern web browser (Chrome, Firefox, Edge, Safari) to run this application.

### Installation

1.  **Clone the repository** (or download the source code):
    ```bash
    git clone <repository-url>
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd task_manager
    ```

### Running the App

Simply open the `index.html` file in your web browser.

- **Option 1**: Double-click `index.html` in your file explorer.
- **Option 2**: Right-click `index.html` -> Open with -> Google Chrome (or your preferred browser).
- **Option 3**: If you are using VS Code, you can use the "Live Server" extension to serve the file.

## ğŸ•¸ï¸ Usage

1.  **Add a Task**: Type your task in the input field and click "Add" or press **Enter**.
2.  **Complete a Task**: Click on any task item to mark it as done. Listen for the sound effect!
